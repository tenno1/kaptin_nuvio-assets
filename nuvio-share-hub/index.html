How To Install & Import

A simple step-by-step visual walkthrough to get custom folder designs running in seconds.

Step 01 / 04

Get Your Tools Ready

Before importing custom collections, make sure you have the basics ready.

  • ✓
    Stremio Installed: Ensure you have the Stremio app active on your PC, Android, or web browser.
  • ✓
    Nuvio Configured: Verify you have the Nuvio addon installed or are utilizing Nuvio Studio to edit layouts.
  • ✓
    Stable Internet: Custom cover images and hover gifs are hosted on GitHub CDN and load dynamically.
Step 02 / 04

Choose Your Collection Style

You have two main paths to load the collections, depending on what device you use:

Method A: Download JSON

Perfect for PC and Web users. Scroll to the Collection Explorer below, click Download JSON to grab files directly to your device, and import them locally.

Method B: Copy Link

Perfect for seamless remote importing. Click Copy Direct Link. This copies the absolute raw URL directly from GitHub. You can paste this URL right into Nuvio’s import configuration page.

Step 03 / 04

Import into Nuvio Studio

Import the JSON files into Nuvio to populate your folders:

Important: Overwriting existing configurations will clean your current list. If you want to merge collections, ensure you append or upload individual files one by one.
  1. Open your **Nuvio Studio** manager tab.
  2. Navigate to the **JSON Configuration** import settings.
  3. For Method A: Click *Choose File*, select your downloaded `.json` file, and hit *Upload*.
  4. For Method B: Paste the copied GitHub link into the *Import from URL* field, then click *Fetch & Import*.
Step 04 / 04

Verify & Enjoy Custom Designs

Excellent! Your collections are imported. Now double-check that they load properly:

You're All Set!

Launch Stremio and check your folders. If hover cards do not react immediately, ensure "GIF Hover Effects" is enabled in Nuvio settings, and clear your Stremio cache if necessary.
